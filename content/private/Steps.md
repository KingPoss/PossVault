1. Create ticket
2. Ask name, phone number, location
3. Gather description from user's explanation, near verbatim
4. generate short desc from description
5. Attempt to troubleshoot
6. If FCR, close ticket
7. If not, assign out, explain why, timeframe, and ask if there is anything else we can do to help
8. Always end with have a lovely day, let them know we can be reached 24/7
